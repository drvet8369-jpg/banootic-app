## Style Guidelines:

- Primary color: Soft mint green (#B5E2BF), inspired by the feeling of natural freshness and subtle elegance, reminiscent of traditional Iranian art, will create a soothing user experience.
- Background color: Very light mint (#E8F8EF), almost white, maintaining a clean and airy aesthetic.
- Accent color: Dusty rose (#D6A2A8), an analogous color adding a touch of femininity and contrast to highlight key elements without being overly bold.
- Font: 'Alegreya', a humanist serif for an elegant, intellectual, contemporary feel. 
- Use a grid-based layout to ensure a clean, organized and responsive design.  The layout should prioritize readability and ease of navigation.
- Employ subtle, hand-drawn-style icons to represent each service category, contributing to the overall feminine and approachable aesthetic.
